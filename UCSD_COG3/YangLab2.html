How to define a table?

To define a table in HTML, use the <table> element. Within this element, use the following HTML elements:

  • <tr>: Defines a table row.
  • <th>: Defines a header cell in a table.
  • <td>: Defines a standard cell in a table.
  • <thead>: Groups the header content in a table.
  • <tfoot>: Groups the footer content in a table.
  • <caption>: Provides a title or a caption for the table.

What is the difference between colspan and rowspan?

colspan allows a cell to span multiple columns, whereas rowspan allows a cell to span multiple rows.

What is the difference between cell spacing and padding?

Cell spacing is the space between individual table cells, whereas padding is the space between the cell content and its borders.

How do you define table borders using CSS?

You can define table borders using CSS with the border property. To control border widths, use the border-width property.
