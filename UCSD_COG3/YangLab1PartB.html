Questions and Answers

Q1: What is DOCTYPE Statement?
DOCTYPE, or Document Type Declaration, is a code instruction that tells a web browser what version of HTML a page is written in. It's the first line of code required in every HTML or XHTML document.
Q2: How do you use: <ol>, <ul>, <dl>?
<ol>: Ordered List. It creates a list of items in a specific order, usually numbered.
<ul>: Unordered List. It creates a list of items in no particular order, usually bulleted.
<dl>: Definition List. It creates a list of terms and their definitions.
Q3: Difference between text editor and word processor?
A text editor is a program used for editing plain text files, while a word processor is a software application with advanced features for formatting text, images, and other content, typically for document creation and editing.
Q4: What is the client-server model?
The client-server model is a distributed application structure that partitions tasks between servers, which provide resources or services, and clients, which request them. It is the foundational structure of many network applications.
Q5: Network Protocols:
FTP: File Transfer Protocol. A standard network protocol used for the transfer of computer files between a client and server on a computer network.
HTTP: HyperText Transfer Protocol. An application protocol for distributed, collaborative, hypermedia information systems, and the foundation of data communication on the World Wide Web.
TCP/IP: Transmission Control Protocol/Internet Protocol. A set of networking protocols that allows two or more computers to communicate and share data across a network.
